## Методология оценки

Каждая генерация автоматически оценивается по трём измерениям:

- **Соответствие схеме (0-100%)**: Валидация JSON схемы и наличие обязательных полей
- **Качество контента (0-100%)**: Глубина, полнота, структура и педагогическая ценность
- **Качество языка (0-100%)**: Естественность, грамматика, терминология, отсутствие placeholders

**Общий балл** = Среднее значение всех трёх измерений
