## Технические замечания

### Проблемы с JSON-форматом

| Модель | Проблема | Рекомендация |
|--------|----------|--------------|
| Devstral 2512 | Смешивает `\"` и fancy quotes `"` | Добавить post-processing для нормализации кавычек |
| Nemotron Nano 30B | Обрезает ответы / неверный JSON | Не использовать для структурированной генерации |

### Рекомендации по промпту

Для улучшения JSON-output:
1. Добавить явное указание: `Use only standard ASCII quotes (\") in JSON`
2. Увеличить `max_tokens` для длинных ответов
3. Использовать JSON Schema validation в промпте
